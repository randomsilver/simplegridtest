    Chrome DevTools: Advantages over Firebug

    • Chrome DevTools are native to the browser
    • Built-in CPU Profiler and Heap Snapshot profiler
    • Built-in Audit tool
    • Can access/edit LocalStorage, SessionStorage, Cookies, SqlLite DB's etc. from Resources panel
    • Built-in JSON and XML viewers
    • Built-in JavaScript Syntax highlighting
    • JavaScript debugger has more features (WebWorker/XHR/DOM/Event breakpoints
    • JavaScript debugger shows an element's listeners and JavaScript file/line location
